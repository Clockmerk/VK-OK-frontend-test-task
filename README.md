## Purpose of the application
Тестовое задание реализации компонентов счетчика и кнопки
Задание указано в [Task.md](Task.md)
<br>

## Getting Started

```
npm i
npm run dev

```

## Storybook

```
npm run storybook

```

## MVP functionality

- Компонент кнопки
- Компонент счетчика
- Обзор компонентов с помощью storybook

## Stack

- React
- TS
- Stylus
- Storybook
- Jest
- react-testing-library
